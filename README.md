A collection of web projects built with modern technologies.

## Projects

### 1. CivicAlert

A civic reporting application that allows users to report issues without fear. Features a clean, accessible interface with real-time map integration.

**Tech Stack:**
- HTML5
- CSS3 (Custom properties, modern styling)
- JavaScript
- Leaflet.js (Real-time maps)
- Google Fonts (Syne, Instrument Sans)

**Features:**
- Interactive map view
- Issue reporting system
- Responsive design
- Clean, green-themed UI

**To run:** Open `index.html` in a web browser.

---

### 2. Antigravity Bot

A modern AI chatbot interface powered by Google Gemini API. Features a sleek dark theme with a responsive sidebar and chat functionality.

**Tech Stack:**
- HTML5
- CSS3 (Dark theme with accent colors)
- JavaScript
- Google Fonts (Inter)
- Google Gemini API

**Features:**
- AI-powered chatbot
- Dark theme UI with accent colors
- Chat history
- Responsive sidebar
- Markdown-style formatting in responses
- Conversation context preservation

**API Configuration:**
The bot uses the Google Gemini API. Update the API key in `antigravity_bot/script.js`:
```JavaScript
const API_KEY = "YOUR_API_KEY";
```

**To run:** Open `antigravity_bot/index.html` in a web browser.

---

## Getting Started

1. Clone or download this repository
2. Navigate to the project folder you want to run
3. Open the respective `index.html` file in your browser
